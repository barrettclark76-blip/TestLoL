How it works: This UI connects to a Random Forest classifier trained on ranked match data. Enter your 4 teammates' picks and all 5 enemy picks, select your role, then click Analyse Draft. The model returns champions ranked by win-score — the probability that a winning team played that champion in a similar draft context.

Setup: The recommender requires a running Python backend serving the model via a local API. See server.py for the Flask endpoint, or replace API_URL below with your deployed endpoint. For local use, run python server.py and open this page via a local server.
